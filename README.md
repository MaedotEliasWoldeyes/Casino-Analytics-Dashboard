##  Overview
This project analyzes casino player behavior, betting activity, and table performance using SQL and Tableau. The project transforms raw transactional data into meaningful KPIs and visual insights to support business decision-making around player engagement and revenue patterns.

---

##  Tools Used
- SQL (data analysis, joins, aggregation)
- Tableau (dashboard visualization)
- Excel (data preparation)

---

##  Dataset Description
The analysis is based on relational casino datasets:
- Players data
- Bets transactions
- Sessions data
- Table activity data

These datasets were connected using relational keys to enable cross-analysis of player and betting behavior.

---

##  Key KPIs
- Total bets placed
- Average bet amount per player
- Player session frequency
- Table performance activity
- Betting behavior trends over time

---

##  Dashboard Features
- Interactive KPI dashboard for casino performance
- Player-level behavioral analysis
- Table usage and engagement tracking
- Filters for time, player activity, and betting patterns

---

##  Key Insights
- Identified variations in betting behavior across players
- Found high-activity tables with consistent engagement
- Observed peak usage patterns across sessions
- Highlighted differences in player spending levels
